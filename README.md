## Features

*   **Authentication**: Secure Sign Up and Login using Firebase Auth.
*   **Real-time Messaging**: Instant message delivery using Firestore.
*   **User Profiles**: Customizable profiles with image uploads via Firebase Storage.
*   **Responsive UI**: Clean, programmatic UI with smooth keyboard handling.
*   **Search**: Search for users to start a new conversation.

## Tech Stack

*   **Framework**: UIKit (Programmatic UI & Auto Layout)
*   **Language**: Swift 5
*   **Architecture**: MVVM (Model-View-ViewModel)
*   **Backend**: Firebase (Auth, Firestore, Storage)
*   **External Libraries**:
    *   `GoogleSignIn` – Seamless social authentication.
    *   `GoogleMaps` – Interactive maps integration.
    *   `SDWebImage` – Asynchronous image loading and caching.
    *   `ImageSlideshow` – Elegant image galleries and carousels.
    *   `SwiftAudioPlayer` – Comprehensive audio streaming and playback.
    *   `JGProgressHUD` – Lightweight progress indicators (loaders).
    *   `IQKeyboardManagerSwift` – Automatic keyboard handling.
